# 常见问题

## 工作原理

:::: info 原理解析
本程序利用了游戏引擎的渲染机制和 Windows 系统的窗口管理能力。
::::

### 1. 渲染机制

- 在多数使用现代游戏引擎（如 UE4/5、Unity 等）开发的游戏中，窗口模式或无边框窗口模式下的渲染分辨率通常由窗口尺寸决定。

  ```ts
  // UE4/5 引擎示例
  r.ScreenPercentage // 控制实际渲染分辨率与窗口分辨率的比例
  ```

- 当窗口尺寸调整时，游戏引擎会根据新的窗口尺寸重新计算渲染分辨率。
- 即使窗口尺寸超出显示器物理大小，游戏仍可能按照实际窗口大小完成完整渲染。
- 因此，在《无限暖暖》中将 **拍照 - 照片画质** 设置为 **窗口分辨率** 时，可以输出随窗口分辨率提升的照片。

### 2. 窗口管理

- 本程序通过 Windows 标准窗口管理 API 调整窗口尺寸、位置和样式。

  ```cpp
  // Windows API 核心调用
  SetWindowPos() // 调整窗口大小和位置
  WS_POPUP       // 需要时切换为无边框样式
  ```

- 这些操作属于系统层面的标准窗口管理行为，例如系统分辨率变化、屏幕旋转或多显示器窗口移动时的窗口重排。

### 3. 安全性

:::: danger 程序不会：
- 修改游戏内存
- 注入游戏进程
- 修改游戏文件
::::

## 下载时被杀毒软件拦截

网页下载可能被浏览器或安全软件拦截。建议使用 **百度网盘** 下载，链接可在 [安装与运行](/guide/getting-started) 页面找到。

## 安装时提示安装失败

点击错误提示中的 **查看日志文件**，将日志内容复制到豆包等 AI 工具中询问解决方案。日志内容因环境而异，AI 分析是最快的排查方式。

## 主界面白屏或黑屏

大概率是资源文件缺失。请确保便携版解压时，将压缩包内的 **所有文件** 解压到一个单独的文件夹中，而不是只解压部分文件。

## 主界面窗口无法拖拽

请更新 [WebView2 Runtime](https://developer.microsoft.com/zh-cn/microsoft-edge/webview2)。本程序的主界面依赖 WebView2 运行，版本过旧可能导致交互异常。

## 《无限暖暖》拍照后分辨率或比例与预期不符

请进入游戏设置，确认以下两项：
- **显示模式** 为 **窗口模式**
- **拍照 - 照片画质** 设置为 **窗口分辨率**（非 4K 等固定档位）

## 如何卸载

确认完全退出程序后：
- **安装版**：打开 **系统设置 > 应用 > 应用和功能**，找到 SpinningMomo 后点击卸载
- **便携版**：直接删除整个程序文件夹
