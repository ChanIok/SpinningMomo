# High-Res Screenshots

## In-Game Camera (Recommended)

For *Infinity Nikki*, we recommend using the in-game **Momo's Camera** for photos. The game pauses rendering when the camera is open, avoiding motion blur and temporal artifacts. It also saves photo metadata alongside the image.

Click **Game Album** on the floating window to jump straight to the game's album folder — no need to hunt for it manually.

## Built-In Screenshot

Default hotkey: `F11`

The built-in screenshot captures the game window as-is and saves the file to `SpinningMomo` under your system **Videos** folder (configurable in Settings).

::: info When to use it
- When the in-game camera won't save at your current ultra-high resolution
- When you need to capture other games or windows

For normal *Infinity Nikki* photography, you probably won't need this.
:::
